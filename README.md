# Zen Kaplumbağa: Sonsuz Ada Bahçesi 🐢🌸

Kız arkadaşın için özel olarak tasarlanmış; yolda giderken, uzanırken veya dinlenirken tek elle rahatça oynayabileceği, sakinleştirici, sonsuz seviyeli bir web bulmaca oyunu.

## ✨ Özellikler
- **Sonsuz Seviye (Procedural Generation):** Her seviye, algoritmik olarak benzersiz kombinasyonlarla üretilir. Seviyeler ilerledikçe tatlı bir şekilde ölçeklenir.
- **Göz Yormayan Pastel Tema:** Uzanırken veya loş ışıkta gözü yormayacak soft renk paleti, parlayan lotus çiçekleri ve minimal çizgiler.
- **Kafa Boşaltma Mekaniği:** Zaman sınırı, can sınırı veya "kaybetme" stresi yoktur. Sadece adadaki su yollarını döndürerek birbirine bağlar. Su yolları birleştikçe adada çiçekler açar.
- **Sevimli Kaplumbağa Animasyonları:** Su yolları birleştikçe kaplumbağa akış yönüne doğru tatlı floper hareketleriyle yüzerek ilerler.
- **Web Audio API Ses Sentezi:** Dışarıdan büyük müzik/ses dosyaları yüklemeye gerek kalmadan, kod içerisinden tamamen matematiksel olarak üretilen soft su damlası ve pentatonik zen çan sesleri içerir.

## 🚀 Canlıya Alma (Render / GitHub Pages)
Bu proje sıfır konfigürasyon (Zero-Dependency) ile çalışır. Canlıya almak için:
1. Bu repo içerisindeki dosyaları GitHub hesabına yükle.
2. **Render.com** üzerinden yeni bir **Static Site** oluşturup bu repoyu bağla YA DA doğrudan repo ayarlarından **GitHub Pages**'i aktif et.
3. Paylaşılan link üzerinden telefon, tablet veya bilgisayar tarayıcısından anında oynanabilir!
